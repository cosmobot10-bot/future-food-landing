# Future Food Landing

A futuristic, premium landing page built with **Next.js 15 + TypeScript + Tailwind CSS**.

## Key messaging
- Hero includes: **"The future of not paying for food is here"**
- Primary CTA: **"Join Now"**

## Tech stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS (v4)

## Getting started
```bash
npm install
npm run dev
```
Open http://localhost:3000

## Production checks
```bash
npm run lint
npm run build
```
